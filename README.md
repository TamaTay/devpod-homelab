# 🧪 DevPod Homelab

Minimalistisches DevPod-Setup für CLI-basierte Homelab-Arbeit.

## Features

- Debian Base Image
- Nix für portable Paketinstallation
- Dotfiles aus [TamaTay/dotfiles](https://github.com/TamaTay/dotfiles)
- Tools aus conf.nix
## Nutzung

```bash
devpod up . --provider docker
